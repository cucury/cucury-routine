# 쿠쿠리 루틴

기본적으로 루틴을 만드는 것으로 진행하는 프로젝트 입니다.

프로젝트 참석자는 총 5명으로
메인 기여자가 대부분 기여하지만
나머지 인원은 기여하고 싶은 만큼 기여하면 되므로
부담없이 진행하는 것을 목표로 하고 있습니다.

- 프로젝트의 특징
  - 처음에는 아--주 단순한 기능만 만들어 본다(복잡한 기능 만들지 말기!)
  - 기여도 0.00001%만 이상이라면 문제 없음
